🚀 Base Startup Landing Page

A modern, high-performance landing page built entirely with Tailwind CSS. This project demonstrates UI implementation skills as part of my Bootcamp portfolio.

---
🌐 Live Demo: https://tailwind-startup.vercel.app/

⚙️ Tech Stack

HTML5 – Semantic structure for accessible markup  

Tailwind CSS – Utility-first framework for rapid UI styling  

Vanilla JavaScript – Lightweight client-side interactions  

External JavaScript Libraries – Used for scroll animations and visual effects

🗝️ Key Features

Fully Responsive: Optimized for Mobile, Tablet, and Desktop.

Modern Components: Hero section, Pricing tables, Testimonials, and Contact form.

Performance: Optimized CSS bundle using Tailwind JIT engine.

Layout: Advanced layouts using Flexbox and CSS Grid.

👨‍💻 Quick Start

Clone the repo: git clone https://github.com/DucNguyen1402-dev/tailwind-startup-template.git
cd project
npm install
npm run dev
